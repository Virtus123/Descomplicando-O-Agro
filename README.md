# Descomplicando o Agro

Site educativo sobre práticas agrícolas sustentáveis — projeto desenvolvido para a **Olimpíada Agrinho** de programação.

## Conteúdo

- Página principal com navegação temática
- Módulos sobre irrigação, pragas e compostagem
- Layout responsivo com HTML5 e CSS3

## Tecnologias

- HTML5
- CSS3

## Visualizar

Clone o repositório e abra `index.html` no navegador.
